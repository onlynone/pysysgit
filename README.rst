===============================
System Git
===============================

.. image:: https://img.shields.io/pypi/v/pysysgit.svg
        :target: https://pypi.python.org/pypi/pysysgit


Utility to call git from python. Examples::

  >>> from sysgit import git
  >>> git('ls-files')

* Documentation: https://pysysgit.readthedocs.org.
